gcc 编译源文件 
`gcc -m32 a.cpp -o a.exe`

-m32  编译输出32可执行文件

-o 指定输出的可执行文件名

```js
#include <iostream>

int main(int argc, char** argv){
    std::cout << argc << std::endl;
    for(int i = 0;i < argc;i++){
        std::cout << argv[i] << std::endl;
    }
    return 0;
}
```
